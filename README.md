## Лабораторная работа "Сверточные нейронные сети"

## Структура репозитория

- `laba 3 ДЗ_Lite.ipynb` –  ноутбук для задания Lite: создание нейронной сети для решения задачи бинарной классификации
- `laba 3 ДЗ_Pro.ipynb` – ноутбук для задания Pro: обучить счерточную сеть с заданными гиперпараметрами распознавать рукописные цифры
- `laba 3 ДЗ_Ultra_Pro.ipynb` – ноутбук для задания Ultra Pro: создать рукописную модель для распознования автомобилей и сравнить ее с преодобученной моделью
- `Pytorch laba 3.ДЗ Lite.ipynb` – ноутбук для задания Lite:создание нейронной сети для решения задачи бинарной классификации. Выполнено с использованием Pytorch

## Требования

Для запуска ноутбуков рекомендуется использовать **Google Colab**. Все необходимые библиотеки предустановлены в Colab.
